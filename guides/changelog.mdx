## February 11, 2026

### AntiGravity Support

- Added first-class AntiGravity setup documentation at `/ai-integrations/antigravity`.
- Clarified the Streamable HTTPS MCP config (`serverUrl` only) and auth recovery flow.
- Documented practical troubleshooting for repeated auth prompts and `invalid_token` scenarios.

### Tracking Setup V3 Flow

- Documented the expanded setup journey states:
  - `choose_objective`
  - `choose_events`
  - `confirm_event_plan`
- Documented new tracking setup inputs:
  - `tracking_objective`
  - `selected_standard_events`
  - `selected_custom_events`
  - `confirm_event_plan`
  - `auto_create_custom_conversions`
- Clarified setup outputs used by clients, including `required_inputs`, event-plan fields, and verification hints.

### Tracking Audit Tool

- Added docs for `meta_tracking_setup_audit` (read-only).
- Covered health checks, issue summaries, event-level drift detection, and AI-ready remediation output.
- Added recommended troubleshooting sequence starting with setup audit, then targeted follow-up tools.

### Auth and Permission Hardening

- Clarified tracking setup requirement for tracking-ready Meta auth (system-user style, non-expiring token).
- Documented permission-recovery behavior when Pixel creation needs additional Meta permissions.

### Docs Routing Updates

- Added docs compatibility routing support for:
  - `/docs/setup/antigravity`
  - `/docs/setup/windsurf`
  - `/docs/changelog`

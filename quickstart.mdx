title: "Quickstart"
---

Follow these steps to connect your AI assistant and start with either analytics or tracking.

<Steps>
  <Step title="Create an AdsGateway account">
    Go to [adsgateway.io](https://adsgateway.io) and sign up.
  </Step>

  <Step title="Connect your AI assistant">
    Pick an integration and complete MCP connection:

    <CardGroup cols={2}>
      <Card title="AntiGravity" href="/ai-integrations/antigravity">
        Streamable HTTPS MCP setup
      </Card>
      <Card title="Cursor" href="/ai-integrations/cursor">
        One-click or manual MCP setup
      </Card>
      <Card title="Lovable" href="/ai-integrations/lovable">
        Custom MCP connector setup
      </Card>
      <Card title="Claude (Coming Soon)" href="/ai-integrations/claude">
        Planned integration
      </Card>
      <Card title="Windsurf (Coming Soon)" href="/ai-integrations/windsurf">
        Planned integration
      </Card>
      <Card title="ChatGPT (Coming Soon)" href="/ai-integrations/chatgpt">
        Planned integration
      </Card>
    </CardGroup>
  </Step>

  <Step title="Connect Meta in AdsGateway">
    Authenticate and authorize Meta so your assistant can run analytics and tracking tools.
  </Step>

  <Step title="Choose your workflow">
    **Option A — Analytics**

    - List/select account
    - Run read-only analysis and reports

    **Option B — Tracking (Pixel + CAPI)**

    - Run `setup_meta_tracking` with `site_url`
    - Complete confirmations and create site + key
    - Verify with test event and setup audit
  </Step>

  <Step title="Verify success">
    Analytics verification:

    ```text
    List my Meta ad accounts
    ```

    Tracking verification:

    ```text
    Audit tracking setup for site_id "<site_id>" over the last 24 hours
    ```
  </Step>
</Steps>

<Note>
  Analytics remains read-only. Editing/updating campaigns/ads and ad-writing are still Coming Soon.
</Note>

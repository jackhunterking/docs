## MCP Server URL

All integrations use the same server URL:

```text
https://ads-gateway.onrender.com/mcp
```

<Note>
  Your ad account connections and tracking configuration are tied to your AdsGateway account, so you can use the same setup across supported AI assistants.
</Note>

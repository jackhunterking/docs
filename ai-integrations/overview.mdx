## What is MCP?

MCP (Model Context Protocol) is an open standard that allows AI assistants to securely connect to external tools like AdsGateway.

<Info>
  All AI integrations use the same AdsGateway server. Your ad account connections work across any AI assistant you set up.
</Info>

## Quick Setup (Cursor)

<Steps>
  <Step title="Open Cursor Settings">
    Go to **Settings** > **MCP** in Cursor
  </Step>
  <Step title="Add AdsGateway Server">
    Add the server URL: `https://ads-gateway.onrender.com/mcp`
  </Step>
  <Step title="Authenticate">
    Log in when prompted to connect your AdsGateway account
  </Step>
  <Step title="Start Using">
    Ask questions like "What was my ad spend yesterday?"
  </Step>
</Steps>

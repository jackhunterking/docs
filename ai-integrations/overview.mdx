## What is MCP?

MCP (Model Context Protocol) is an open standard that allows AI assistants to securely connect to external tools like AdsGateway.

<Info>
  All AI integrations use the same AdsGateway server. Your ad account connections work across any AI assistant you set up.
</Info>

## Quick Setup

<Steps>
  <Step title="Choose your AI assistant">
    Click on your preferred AI assistant above to see the setup guide
  </Step>
  <Step title="Add the AdsGateway server">
    Each guide shows you exactly how to configure the MCP server
  </Step>
  <Step title="Authenticate">
    Log in when prompted to connect your AdsGateway account
  </Step>
  <Step title="Start using">
    Once connected, you can query your ad data immediately
  </Step>
</Steps>

## Server URL

All AI integrations use the same server URL:

```
https://ads-gateway.onrender.com/mcp
```

<Note>
  You don't need to remember this URL - each integration guide includes the complete configuration.
</Note>

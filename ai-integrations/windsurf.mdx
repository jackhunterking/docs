Windsurf is a modern AI-powered development environment with MCP support. This guide shows you how to connect AdsGateway to Windsurf.

## Prerequisites

- [Windsurf](https://codeium.com/windsurf) installed on your computer
- An [AdsGateway account](https://adsgateway.io)

## Manual Setup

If you prefer to configure manually, follow these steps:

<Steps>
  <Step title="Open Windsurf Settings">
    Open Windsurf and go to **Settings** (Cmd/Ctrl + ,)
  </Step>
  
  <Step title="Navigate to MCP Settings">
    In the settings, find the **MCP** or **Extensions** section.
  </Step>
  
  <Step title="Add AdsGateway Server">
    Add a new MCP server with the following configuration:
    
    ```json
    {
      "mcpServers": {
        "Ads Gateway": {
          "url": "https://ads-gateway.onrender.com/mcp"
        }
      }
    }
    ```
  </Step>
  
  <Step title="Enable the Server">
    Toggle the AdsGateway server to **ON**.
  </Step>
  
  <Step title="Authenticate">
    When you first enable the server, Windsurf will open your browser for authentication.
    
    1. Log in to your AdsGateway account (or create one)
    2. Authorize the connection
    3. You'll be redirected back to Windsurf
    
    <Check>
      Look for the "Connected" status in Windsurf's MCP panel
    </Check>
  </Step>
</Steps>

## Verify the Connection

Once connected, try these commands in Windsurf's AI chat:

```
List my Meta ad accounts
```

You should see a list of your connected ad accounts.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Server shows as disconnected">
    1. Check your internet connection
    2. Try toggling the server OFF and ON again
    3. If the issue persists, restart Windsurf
  </Accordion>
  
  <Accordion title="Authentication window doesn't open">
    1. Check if pop-ups are blocked in your browser
    2. Try manually visiting [adsgateway.io](https://adsgateway.io) and logging in
    3. Then toggle the MCP server off and on again
  </Accordion>
  
  <Accordion title="'Authentication required' error">
    Your session may have expired. Toggle the server off and on to re-authenticate.
  </Accordion>
  
  <Accordion title="No tools available">
    1. Make sure the server is toggled ON
    2. Check that authentication completed successfully
    3. Try restarting Windsurf
  </Accordion>
</AccordionGroup>

# Windsurf Setup

Windsurf is a modern AI-powered development environment with MCP support. This guide shows you how to add AdsGateway.

## Prerequisites

- [Windsurf](https://windsurf.ai) installed
- An [AdsGateway account](https://adsgateway.io)

## Setup Steps

<Steps>
  <Step title="Open Windsurf Settings">
    Open Windsurf and access the settings:
    
    - **macOS**: Cmd + ,
    - **Windows/Linux**: Ctrl + ,
    
    Or use the menu: **Windsurf > Settings**
  </Step>
  
  <Step title="Navigate to MCP Configuration">
    In the settings, find the **MCP** or **Extensions** section for Model Context Protocol servers.
  </Step>
  
  <Step title="Add AdsGateway Server">
    Add a new MCP server with the following configuration:
    
    ```json
    {
      "mcpServers": {
        "ads-gateway": {
          "url": "https://ads-gateway.onrender.com/mcp"
        }
      }
    }
    ```
    
    <Tip>
      The configuration file location may vary by platform. Check Windsurf's documentation for the exact path.
    </Tip>
  </Step>
  
  <Step title="Enable the Server">
    Make sure the AdsGateway server is enabled in your MCP settings.
  </Step>
  
  <Step title="Authenticate">
    When prompted, complete the authentication flow:
    
    1. Click the login link that appears
    2. Sign in to your AdsGateway account
    3. Authorize the connection
    4. Return to Windsurf
    
    <Check>
      You should see AdsGateway tools available in Windsurf's AI assistant
    </Check>
  </Step>
</Steps>

## Verify the Connection

Test the connection by asking Windsurf's AI:

```
List my Meta ad accounts
```

You should see a list of your connected advertising accounts.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Server not connecting">
    1. Verify your internet connection
    2. Check that the server URL is correct
    3. Restart Windsurf
  </Accordion>
  
  <Accordion title="Authentication loop">
    If you keep being asked to authenticate:
    1. Clear Windsurf's cache
    2. Log in at [adsgateway.io](https://adsgateway.io) directly
    3. Then retry in Windsurf
  </Accordion>
  
  <Accordion title="Tools not appearing">
    1. Make sure the MCP server is enabled
    2. Check the configuration syntax
    3. Restart Windsurf after configuration changes
  </Accordion>
</AccordionGroup>

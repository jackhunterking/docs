## What is Claude?

Claude is Anthropic's AI assistant, available as Claude Code and Claude Desktop.

## Capability Model

When available, this integration supports both AdsGateway workflows:

- **Analytics**: read-only ad analysis, reports, and creative analysis
- **Tracking**: Pixel + Conversions API setup, verification, and health audit

## MCP Installation Path (When Enabled)

When Claude MCP support is enabled in your environment, install AdsGateway MCP with:

- Transport URL: `https://ads-gateway.onrender.com/mcp`
- Server name: `Ads Gateway`

## Connecting Your Account

<Steps>
  <Step title="Set Up Your AI Assistant">
    First, connect AdsGateway to your AI assistant:
    - [Cursor](/ai-integrations/cursor)
    - [Claude](/ai-integrations/claude)
    - [Lovable](/ai-integrations/lovable)
    - [ChatGPT](/ai-integrations/chatgpt)
  </Step>
  
  <Step title="Request Meta Connection">
    In your AI assistant, type:
    
    ```
    List my Meta ad accounts
    ```
    
    If you haven't connected Meta yet, you'll be prompted to authenticate.
  </Step>
  
  <Step title="Authorize with Facebook">
    Click the authorization link to connect:
    
    1. Log in to Facebook (if not already logged in)
    2. Review the permissions AdsGateway is requesting
    3. Select which ad accounts to connect
    4. Click "Continue" to authorize
    
    <Warning>
      Make sure you have admin access to the ad accounts you want to connect.
    </Warning>
  </Step>
  
  <Step title="Select Your Account">
    Once connected, select which account to use:
    
    ```
    Select the account "My Business Account"
    ```
    
    <Check>
      You're now ready to query your Meta data!
    </Check>
  </Step>
</Steps>

## Available Tools

AdsGateway provides Meta tools grouped into analytics, journey reports, and creative analysis.

<Info>
  Analytics tools are read-only: we cannot create, edit, pause, or delete campaigns/ads. See <a href="/feature-availability">Feature Availability</a> for what’s live vs coming soon.
</Info>

### Analytics (Read-Only)

#### Account Management

| Tool | Description | Example Prompt |
|------|-------------|----------------|
| `meta_accounts_list` | List all connected ad accounts | "List my Meta ad accounts" |
| `meta_accounts_select` | Set the active account | "Select account act_123456789" |
| `meta_context_get` | Check which account is active | "Which account am I using?" |

#### Campaign Data

| Tool | Description | Example Prompt |
|------|-------------|----------------|
| `meta_campaigns_list` | List campaigns with filters | "Show me active campaigns" |
| `meta_adsets_list` | List ad sets in a campaign | "List ad sets in my Summer Sale campaign" |
| `meta_ads_list` | List ads in an ad set | "Show me all ads" |
| `meta_ad_get` | Get detailed ad info with creative | "Get details for ad 123456" |
| `meta_insights_get` | Get performance metrics | "What was my spend last week?" |

#### AI Analysis

| Tool | Description | Example Prompt |
|------|-------------|----------------|
| `meta_analysis_summary` | Account performance overview | "Give me an account summary" |
| `meta_analysis_top_movers` | Campaigns with biggest changes | "Which campaigns improved most?" |
| `meta_analysis_anomalies` | Detect unusual patterns | "Are there any anomalies?" |

### Journey Reports

These tools are designed to answer “how is this performing?” in **one call**, for a specific date range.

| Tool | Description | Example Prompt |
|------|-------------|----------------|
| `ads_campaign_report` | Campaign report for a date range (optional comparison) | "Generate a campaign report for campaign 123 since 2026-02-01 until 2026-02-07" |
| `ads_adset_report` | Ad set report for a date range (optional comparison) | "Generate an ad set report for ad set 456 since 2026-02-01 until 2026-02-07" |
| `ads_ad_report` | Ad deep-dive report (creative metadata + performance) | "Generate an ad report for ad 789 since 2026-02-01 until 2026-02-07" |

## Metrics Available

AdsGateway can retrieve these Meta metrics:

### Core Metrics
- **Spend** - Total amount spent
- **Impressions** - Number of times ads were shown
- **Reach** - Unique users who saw your ads
- **Clicks** - Link clicks and all clicks
- **CTR** - Click-through rate
- **CPC** - Cost per click
- **CPM** - Cost per 1,000 impressions

### Conversion Metrics
- **Conversions** - Total conversions
- **Cost per Conversion** - Efficiency metric
- **Purchase ROAS** - Return on ad spend
- **Add to Cart** - Shopping conversions
- **Leads** - Lead generation conversions

### Video Metrics
- **Video Views** - 3-second and ThruPlay views
- **Video Watch Time** - Average watch duration
- **Video Completion Rate** - Percentage who watched to end

### Engagement Metrics
- **Reactions** - Likes, loves, etc.
- **Comments** - Post comments
- **Shares** - Content shares
- **Page Engagement** - Total engagement

## Permissions & Access

AdsGateway asks for the minimum permissions required for the features you enable.

### Analytics (Read-Only)

For analytics, we request read permissions to access ad account structure and performance insights. We cannot create, edit, pause, or delete campaigns/ads.

<Info>
  You will always see the exact permissions on Meta's authorization screen before approving.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="'Meta not connected' error">
    You need to authorize AdsGateway to access your Meta accounts:
    
    1. Go to your Connections page: [adsgateway.io/dashboard/connections](https://adsgateway.io/dashboard/connections)
    2. Click **Connect Meta**
    3. Complete the Meta authorization flow
    4. Return to your AI assistant and try again
  </Accordion>
  
  <Accordion title="Can't see all my ad accounts">
    AdsGateway can only access accounts you have admin permissions for:
    
    1. Check your Facebook Business Manager
    2. Ensure you have admin access to the desired accounts
    3. Reconnect Meta to refresh your account list
  </Accordion>
  
  <Accordion title="Data seems outdated">
    Meta data has a reporting delay:
    
    - Most metrics are available within 15 minutes
    - Some metrics (like conversions) may take up to 48 hours
    - This is a Meta limitation, not AdsGateway
  </Accordion>
  
  <Accordion title="'No account selected' error">
    You need to select an active account first:
    
    ```
    List my Meta ad accounts
    Select the account "Account Name"
    ```
  </Accordion>
</AccordionGroup>

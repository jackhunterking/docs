## How Connections Work

<Steps>
  <Step title="Choose a Platform">
    Select the advertising platform you want to connect from the list above.
  </Step>
  
  <Step title="Authorize Access">
    You'll be redirected to the platform's login page to authorize AdsGateway. This uses OAuth, the industry standard for secure authorization.
    
    <Info>
      AdsGateway only requests **read access** to your ad accounts. We cannot create, modify, or delete your ads.
    </Info>
  </Step>
  
  <Step title="Select Accounts">
    After authorization, you'll see all ad accounts you have access to. Select which accounts you want to use with AdsGateway.
  </Step>
  
  <Step title="Start Querying">
    Once connected, you can query your ad data through any supported AI assistant.
  </Step>
</Steps>

## Data Access

AdsGateway can access the following data from connected platforms:

| Data Type | Description | Example |
|-----------|-------------|---------|
| **Accounts** | Ad account names, IDs, settings | "My Business Account" |
| **Campaigns** | Campaign names, objectives, budgets | "Summer Sale Campaign" |
| **Ad Sets** | Targeting, placements, schedules | "US 25-44 Interest-based" |
| **Ads** | Creative content, status | "Product Video Ad" |
| **Insights** | Performance metrics | Spend, impressions, clicks, conversions |
